📋 Informations Endpoint

🔗 Endpoint Details
URL: POST /products/upload-color-image/:productId/:colorId
Content-Type: multipart/form-data
Authentication: Required (JWT token)
File Size Limit: 5MB
Allowed Types: JPEG, PNG, WebP
Body Field: image (File)
